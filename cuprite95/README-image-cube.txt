
The Cuprite AVIRIS image cube is not included in the github distribution because it
is quite large.  Get it from the USGS external ftp site

ftp://ftpext.usgs.gov/pub/cr/co/denver/speclab/pub/cuprite

ftp commands:

ftp ftpext.usgs.gov
binary
cd /pub/cr/co/denver/speclab/pub/cuprite
get cuprite.95.cal.rtgc.intel.v
get cuprite.95.cal.rtgc.intel.v.hdr
